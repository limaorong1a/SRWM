run_ablation.py
在新进程里设好 ABLA_* 再 runpath 主脚本，避免环境变量在已加载模块里不生效。变体与论文式命名：

full：全模块（默认）
w_o_tot / w_o_reflect / w_o_world：单维消融
w_o_tot_ref / w_o_tot_wm / w_o_ref_wm：两两关
w_o_all：三模块全关
--three-shot 会设置 IDEA3_FEWSHOT_SHOTS=3，用统一主脚本直接运行 three-shot 设定。

示例：

cd /root/storage-public/alfworld-/paradigm_experiments/消融实验
python run_ablation.py --variant w_o_tot -- --num-tasks 3
python run_ablation.py --variant full --three-shot -- --num-tasks 5
